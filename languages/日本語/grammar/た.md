## Verbs

| Tense               | Suffix       |
| ------------------- | ------------ |
| Present Affirmative | ます         |
| Present Negative    | ません       |
| Past Affirmative    | ました       |
| Past Negative       | ましたでした |

## Nouns

| Tense               | Suffix               |
| ------------------- | -------------------- |
| Present Affirmative | です                 |
| Present Negative    | ではありません       |
| Past Affirmative    | でした               |
| Past Negative       | ではありませんでした |
