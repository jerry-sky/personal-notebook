Guide

Installing the OS

Please see the OS-specific instructions in a separate note. See definitions.

Configuration

Copy this repository to ~/notebooks/personal-notebook using git clone.

Run the setup.sh script from the /config directory of this repository to copy various config files to the system.
See «config».

Configuring the Internet browser

  1. Install the Internet Browser.
  2. Install Bitwarden.
  3. Install Dark Reader.
  4. Log in to all online services
    1. Google
    2. GitHub
    3. WhatsApp
    4. Discord
    5. Facebook Messenger
  5. Change default browser to the Internet Browser.
  6. (Opera-only) — if Opera doesn't play some video content use one of these solutions:
    • Enable DRM to view DRM-content such as Prime Video or Netflix etc.
    • Fix libffmpeg.so

Programs

  1. Install the Cloud desktop app.

  2. Enable Redshift
    1. Install redshift if it is not already installed.
    2. Enable autostart.

  3. Install snap why

  4. Configure Git
    1. Install the credentials manager.
    2. Create a new personal github access token.
    3. At first login provide the newly generated token as the password.
    4. Added commit (and tag) signing:
      1. import GPG keys with gpg --import ‹file with the private key›,
      2. use git config --global user.signingKey ‹key ID› to tell git which key to use for signing,
      3. run git config --global commit.gpgSign true and git config --global tag.gpgSign true to enforce signing every commit and every tag with that key.

  5. Install VS Code
    Using

    apt list -a code-insiders | less

    and

    apt install code-insiders=«version from the list; e.g. “1.48.0-1596120937”»

    to establish what version is currently free of irritating Electron-related issues (e.g. white flashes) and install it.

  6. Install Blender
    1. Download it from the downloads page.

    2. Unpack it:

      tar -xf «archive filename»
    3. Move the contents to /opt/blender.

    4. Link the executable:

      ln -s /opt/blender/blender /usr/bin/blender

    5. Install the addons if necessary.

  7. Install GParted apt install gparted

  8. Install the rest of programs from the software list.

Configuring the user experience

  1. Disable icons on the desktop.

  2. Install themes and cursor packs
    1. Install Canta theme
      1. Important note: If used with sudo the theme will install in /usr/share/themes, not in ~/.themes - this will allow GUI apps (e.g. Files, Terminal) to use that theme when running in the elevated privileges mode.
    2. Install icon themes how-to
      1. Flat remix
      2. Papirus icon theme
      3. Quintom Cursors

  3. Adjust mouse (and touchpad) settings.

  4. Disable system sounds.

  5. Follow the rest of the steps of this section in the OS-specific notes. See definitions.

  6. Optional Setup listening to line-in audio at startup pacmd load-module module-loopback latency_msec=5

Keyboard layouts

  1. Install IBus to allow using non-western keyboard layouts and allow using more than 4 keyboard layouts at once.
    1. Should any issues arise, follow instructions provided in this askubuntu.com answer.

  2. For laptops with no dedicated Menu key you need to swap Control_R with Menu to use the Control_R key as the Menu key:

    xmodmap -e "keycode 105 = Menu"
xmodmap -e "keycode 135 = Control_R"

    To view list of all keycodes use xmodmap -pk. To make this behaviour persistent use config found in this repository.

Fonts
